# GeneticAlgorithmPNG
A genetic algorithm for generating PNG files with a target image. 


# Compile with
`g++ -std=c++17 genticalgorithmpng.cpp -o ga_png`

# To run on a Linux terminal
`./ga_png` 

# Dependencies
Requires that you include std_image_write.h source from: https://github.com/nothings/stb/blob/master/stb_image_write.h
